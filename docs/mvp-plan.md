# Campus AI MVP Plan

## Product Positioning

Campus AI starts as a practical campus tool, then grows into lightweight social discovery.

The first product promise is: help students manage courses, express themselves, organize tags, and use AI to make their campus life easier. Social discovery should feel like a natural result of useful data, not the first thing pushed at users.

## MVP Scope

Must ship in v0.1:

- User profile and public campus space
- Manual course schedule management
- Modular profile blocks
- Tag selection and custom tags
- Public schedule and profile visibility controls
- Rule-based sync discovery
- AI profile helper and lightweight icebreakers

Not in v0.1:

- Full chat
- Native mobile app
- Collaborative rich-text editor
- Classroom recording and transcription
- Photo problem solving
- Full club and activity system
- Machine-learning recommendation

## Main User Flow

1. Create account and profile
2. Add tags
3. Add course schedules
4. Edit modular public profile
5. Review public campus space
6. See possible sync matches
7. Use AI to improve profile or generate an opener
